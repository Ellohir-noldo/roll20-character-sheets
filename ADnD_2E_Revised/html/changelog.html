Announcements

Changes in

  • Added Weapon proficiency advancement macro for Psionics
  • Added Nonweapon proficiency advancement macro for Psionics
  • Added nonproficient attack penalty for Psionics
  • Added fields for number of disciplines, sciences, devotions and defense modes, along with progression macros to calculate the amount based on your level
  • Added autocomplete for the Psionic powers from The Complete Psionics Handbook
  • Added proficiencies from The Complete Psionics Handbook
  • Added The Complete Psionics Handbook to the Sheet Settings Tab and to the Game Settings
  • Added a new roll template for psionic powers, similar to spells. This also handles the power score check and also handle enhanced power, as by The Will and the Way
  • Added notes for Psionic saving throw modifiers vs enchantment/charm spells
  • Added notes for Dexterity saving throw modifier from high Dexterity
  • Added notes foldable notes for saving throw modifiers, so these are always available, even when the Save and Magic Resistance Details field is filled out.
  • Added logic to help get the correct character level when setting Thaco, Weapon prof, Nonweapon prof, and Psionic progression macros
  • Added reminder for setting the Wizard and Priest caster level when adding a spell to the sheet.
  • Updated the UI for the Psionic Core tab
  • Updated the Psionic Core power check to use the 2Epsionic roll template
  • Fixed the titles for the max PSP fields
  • Fixed background color for input fields
